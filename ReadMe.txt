# 🕹️ The Writer’s Game

A narrative 2D game built in **Python (Pygame)**.  
The player explores different chapters, interacts with story fragments, and completes writing challenges to progress.  
Each chapter combines atmosphere, exploration, and emotional storytelling.

## 🎮 Features
- Interactive storytelling with emotional progression  
- Dynamic text and writing challenges  
- Atmospheric design and background music  
- Three main chapters with different mechanics

### 🧠 Technologies Used
- Python 3  
- Pygame
